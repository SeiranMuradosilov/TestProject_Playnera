# TestProject_Playnera

Игра полностью выполнена в UI системе.
Для перемещения объектов используется DOTween.
Анимации нанесения макияжа сделаны с помощью стандартных анимаций Unity.

Главный контроллер GameMain, кнопки и другие активные элементы игры,
через события клика подвязаны к соответствующему функционалу контроллера.
Логика и последовательность нанесения макияжа заложена в классах UseBrushTask и 
UseCreamTask, которые работают посредством корутин.
Большая часть функционала перемещения и анимаций заложена в классе Hand.
Меньшая часть в класах инструментов Brush, Cream, Lipstick и MakeupTool.
Функционал переключения(фейда) элементов макияжа на самом персонаже находится
в классах Character и CharacterMakeupItem.
